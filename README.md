# snGlia-PD
Analyses of public datasets GSE178265 (Kamath, 2020) and GSE157783 (Smajic, 2021) containing snRNA seq data from post-mortem tissue. Files associated with date pre-processing and organization is also included, as are scripts used to generate graphs.
[June 2026 update] following reviewer comments, an updated method was added for pseudobulk analysis of the aggregate expression matrices for the above two datasets and a third dataset GSE184950 (Wang, 2024).
